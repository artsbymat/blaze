Blaze supports ordered and unordered lists also nested lists with the following syntax:

```markdown
1. First item
2. Second item
3. Third item
   - First sub-item
   - Second sub-item
   - Third sub-item
     - First sub-sub-item
     - Second sub-sub-item
     - Third sub-sub-item
```

   - Second sub-item
   - Third sub-item
