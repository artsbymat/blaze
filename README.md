# Blaze - Static Site Generator

A blazing fast Static Site Generator built with Go, featuring hot reload for excellent developer experience.

## Features

- **Fast Markdown Parsing** - Convert markdown to HTML quickly
- **Template-Based Rendering** - Customize your site with Go templates
- **Hot Reload** - Automatic browser refresh on file changes
- **Runtime Configuration** - JSON-based config with hot reload support
- **Simple CLI** - Easy to use command line interface
- **File Watching** - Monitors content, templates, and config changes
